      Welcome to the Code & Circuit Maker Team Powow River Measuring Device Page

      The 2017 Code and Circuit Maker Team class, Caleb Scotch, Mac Mansfield, Brendon Ballon, James Grant, Will Sayles, Ethan Downs, Dane Fawkes, Bill Schaller, and Ken Aspeslagh have created a device that measures the water level & water temperature of the Powwow River in Amesbury.

      Our device uses a rotary encoder, laser distance measurer, and a temperature sensor to measure various types of information about the river. We're using a Particle Electron to wirelessly connect our device to the cell network which then transfers it to a computer at Code & Circuit (which also hosts this very webpage!). This information is then posted here, on the statistics page.
